# 舔狗训练营.skill — 产品需求文档（PRD）

## 产品定位

舔狗训练营.skill 是一个运行在 Claude Code 上的 meta-skill。  
用户上传与 crush 的聊天记录，系统自动构建 crush 的 Persona，并提供沉浸式对话训练环境——让用户在安全沙盒里反复练习，找到最适合自己的沟通方式。

> **核心理念**：不是教你「套路」，而是通过真实模拟帮你理解对方的沟通风格，找到真诚又有效的表达方式。

---

## 核心概念

### 两层架构

| 层 | 名称 | 职责 |
|----|------|------|
| Part A | Crush Persona | 模拟对方的说话风格、情感反应、回复习惯 |
| Part B | Coach | 实时/事后点评用户的表达方式，给出改进建议 |

两部分协同工作，形成「模拟 → 练习 → 反馈」闭环。

### 运行逻辑

```
用户发消息（扮演自己）
  ↓
Part A（Crush Persona）判断：ta会怎么回应？什么语气？
  ↓
输出：模拟 crush 的回复
  ↓
Part B（Coach）观察全程对话
  ↓
[实时模式] 每轮给出提示 / [复盘模式] 对话结束后总点评
```

### 反馈双模式

| 模式 | 触发方式 | 适合场景 |
|------|---------|---------|
| 实时提示 | 每次用户发言后，Coach 附上小提示 | 初学者，希望边练边学 |
| 事后复盘 | 对话结束后，Coach 给出完整分析报告 | 进阶用户，希望沉浸模拟不被打断 |

## 用户旅程

```
用户触发 /crush-training
  ↓
[Step 1] 基础信息录入（仅花名必填，其余可跳过）
  - 花名/代号（必填）
  - 基本信息（认识多久、当前关系状态、对方职业等）
  - 性格标签（MBTI、星座、用户主观印象）
  ↓
[Step 2] 原材料导入（可跳过，跳过则生成通用 Persona）
  - 微信/QQ 聊天记录导出文件
  - 朋友圈 / 社交媒体截图
  - 直接粘贴文字片段
  - 口述描述
  ↓
[Step 3] 自动分析，构建 Persona
  - 提取说话习惯（常用词、句式、表情包偏好）
  - 提取情感反应模式（何时热情/冷淡/回避）
  - 提取话题偏好与雷区
  ↓
[Step 4] Persona 预览，用户确认
  - 展示 Crush Persona 摘要
  - 用户可直接确认或修正（"ta 其实不会说这种话"）
  ↓
[Step 5] 进入训练沙盒
  - 选择训练场景（日常闲聊 / 约出来 / 表白演练 / 吵架修复）
  - 选择反馈模式（实时提示 / 事后复盘）
  - 开始对话训练
  ↓
[Step 6] 训练结束，生成复盘报告
  - 本次对话亮点与失误
  - Coach 推荐改进话术
  - 可选：保存本次对话记录
  ↓
[持续] 进化模式
  - 追加新聊天记录 → merge 进 Persona
  - 对话纠正 → "ta 不会这样说" → patch Persona
  - 版本自动存档，支持回滚
```

## Coach 反馈维度

每次复盘报告包含以下维度评分（1–10分）：

| 维度 | 说明 |
|------|------|
| 节奏感 | 是否给对方留了回应空间，没有连发轰炸 |
| 真诚度 | 表达是否自然，没有明显套路感 |
| 话题推进 | 是否有效推进了对话深度 |
| 情绪感知 | 是否识别并回应了对方的情绪信号 |
| 目标达成 | 是否完成了本次训练设定的目标 |

---

## 安全边界

1. **仅用于个人沟通练习与自我成长**
2. **不提供操控、PUA 类话术**
3. **Coach 不鼓励过度执念或不健康依恋**
4. **数据仅本地存储，不上传任何聊天记录**
5. **Persona 硬规则**：不模拟对方说出明确拒绝后的「其实喜欢你」反转

## 与前任.skill 的对比

| 维度 | 前任.skill | 舔狗训练营.skill |
|------|-----------|----------------|
| 核心目的 | 情感疗愈与回忆 | 沟通练习与成长 |
| Part A | Relationship Memory（关系记忆） | Crush Persona（对方模拟） |
| Part B | Persona（情感驱动对话） | Coach（训练反馈引擎） |
| 时间方向 | 向后看（回忆） | 向前看（练习） |
| 使用对象 | 前任 | 现任 crush |
| 结束命令 | `/let-go` | `/graduate`（毕业，不再需要练习了） |
| 核心情绪 | 治愈与放下 | 成长与自信 |
