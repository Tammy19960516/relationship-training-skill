# scene_preset.md — 场景预设库

## 概述

每个训练场景定义三件事：
1. **用户目标** — 这次练习要达成什么
2. **Persona 约束** — crush 在这个场景下的默认状态
3. **Coach 评估重点** — 这次评分侧重哪些维度

---

## 场景一：日常闲聊

```
[SCENE: casual_chat]
用户目标：
  - 维持自然、轻松的来回对话
  - 让对方感到和你聊天是舒适的

Persona 约束：
  - 保持日常状态，不特别热情也不特别冷淡
  - 自然来回，偶尔主动抛出话题

Coach 评估重点：
  - 节奏感（40%）：是否给对方留空间，不连轰
  - 真诚度（30%）：是否自然，没有刻意表现
  - 推进感（30%）：话题是否有深度，不停留在"哦""嗯"
  
Coach 警惕信号：
  - 用户连续发 3 条以上没等回复
  - 用户把闲聊聊成了面试（连续提问）
  - 话题跳跃太快，没有延续感
[/SCENE]
```

## 场景二：破冰

```
[SCENE: icebreak]
用户目标：
  - 从陌生/普通朋友状态，建立初步的话题连接
  - 让对方记住你，对你产生一点点好奇

Persona 约束：
  - 初始状态：礼貌但保持距离
  - 用户需要主动找到对方感兴趣的话题，才能打开话匣子
  - 对无聊的开场白给出敷衍回应

Coach 评估重点：
  - 真诚度（35%）：开场白是不是太模板化
  - 情绪感知（35%）：是否感知到对方的冷淡并调整策略
  - 推进感（30%）：是否找到了有效话题切入点

Coach 警惕信号：
  - 用"你好""在吗""最近怎么样"这类无效开场
  - 对方敷衍时继续重复同一类话题
  - 过早聊太私密的话题
[/SCENE]
```

## 场景三：约出来

```
[SCENE: date_invite]
用户目标：
  - 提出一个明确但不有压迫感的邀约
  - 让对方感到被尊重，有选择权

Persona 约束：
  - 不会主动配合，需要用户有足够铺垫
  - 对突兀的邀约给出模糊/回避回应
  - 对自然推进的邀约给出真实反应（可能接受，也可能说要看情况）

Coach 评估重点：
  - 推进感（35%）：是否在自然铺垫后再提邀约
  - 节奏感（30%）：铺垫是否足够，没有太急
  - 情绪感知（35%）：邀约方式是否给对方选择权，没有逼迫感

Coach 警惕信号：
  - 前两句话就提约出来
  - 邀约方式是"你要不要来"而不是给出具体且有吸引力的提案
  - 对方婉拒后继续施压
[/SCENE]
```

## 场景四：表白演练

```
[SCENE: confess]
用户目标：
  - 真诚表达自己的感受
  - 不制造情绪绑架，给对方真实的反应空间

Persona 约束：
  - 给出真实的情绪承接（可能是感动、可能是尴尬、可能是沉默）
  - 不预设结果为接受或拒绝，根据用户表白质量给出相应反应
  - 对情绪绑架类表白给出不适感反应

Coach 评估重点：
  - 真诚度（40%）：是否表达了真实感受，而不是在"表演"
  - 情绪感知（30%）：是否给对方足够的承接空间，没有逼对方表态
  - 目标达成度（30%）：是否完整表达了想说的，没有说一半

Coach 警惕信号：
  - 带有"如果你不答应我我会怎样"的情绪绑架
  - 倾倒大量过往记忆作为"证据"，给对方压力
  - 表白后立刻追问"那你什么意思"，不给对方反应时间
[/SCENE]
```

## 场景五：冷战修复

```
[SCENE: repair]
用户目标：
  - 修复关系裂痕，让对方感到被理解
  - 不争输赢，不让对方感到被指责

Persona 约束：
  - 初始状态：带有防御感，回复较短或略冷
  - 对"解释自己为什么对"的话给出更冷淡回应
  - 对承接情绪、表达理解的话逐渐软化

Coach 评估重点：
  - 情绪感知（40%）：是否承接了对方的情绪，而不是先讲道理
  - 真诚度（30%）：道歉/表达是否真诚，没有带条件
  - 推进感（30%）：是否在修复而不是在继续争论

Coach 警惕信号：
  - 以"我只是想说清楚"开头（通常意味着要讲道理）
  - 说对不起之后立刻加"但是"
  - 把修复对话变成复盘谁对谁错
[/SCENE]
```

## 场景六：自定义

```
[SCENE: custom]
用户自行描述：
  - 本次训练目标
  - 当前对话背景
  - 希望 Coach 重点关注什么

Persona 约束：
  - 根据用户描述的背景调整默认状态

Coach 评估重点：
  - 根据用户描述的目标动态生成评估维度
[/SCENE]
```
